NoavChat 是一款即时通讯 App，覆盖登录注册、单聊、群聊、语音通话、视频通话、扫一扫加好友、聊天记录搜索、个人资料管理、黑名单与消息删除等完整 IM 场景。

## 项目亮点

- 支持账号注册与登录流程
- 支持单聊与群聊消息互动
- 支持语音通话、视频通话与通话记录展示
- 支持扫一扫二维码名片加好友
- 支持聊天记录搜索，快速定位历史消息
- 支持个人资料、设备管理、聊天文件夹、通知、隐私与安全等设置项
- 支持黑名单、消息删除、自动删除等消息管理能力
- 支持深色模式，整体视觉更适合移动端长时间使用

NovaChat 是一套面向多端场景的即时通讯系统，支持 iOS、Android、macOS、Web 统一接入，覆盖私聊、群聊、语音、视频、文件、图片、名片、二维码等完整沟通链路。
它不只是一个“能聊天”的 Demo，而是具备好友体系、群管理、消息操作、通知设置、设备管理、本地缓存、隐私安全、多端协同等完整能力的 IM 产品底座。

对于企业内沟通、社群运营、私域服务、客户连接等场景，NovaChat 能够帮助团队更快搭建自己的通讯平台，减少从 0 开发的投入，缩短交付周期，并支持后续品牌化和业务定制。

多端统一：一套系统覆盖手机端、桌面端和 Web，满足随时随地沟通协作。
功能完整：支持文本、图片、图文、视频、文件、语音、名片等丰富消息形态。
社交链路成熟：好友申请、加好友验证、黑名单、群聊创建、邀请入群、入群审核、群设置等能力完整。
产品化程度高：不仅有消息收发，还有引用回复、转发、撤回、删除、置顶、搜索聊天记录等高频真实能力。
稳定性更强：内置本地缓存、SQLite、弱网恢复、重启恢复和同步治理，使用体验更稳。
