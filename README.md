# im-chat
NovaChat 是一套面向多端场景的即时通讯系统，支持 iOS、Android、macOS、Web 统一接入，覆盖私聊、群聊、语音、视频、文件、图片、名片、二维码等完整沟通链路。 具备好友体系、群管理、消息操作、通知设置、设备管理、本地缓存、隐私安全、多端协同等完整能力的 IM 产品。  对于企业内沟通、社群运营、私域服务、客户连接等场景，NovaChat 能够帮助团队更快搭建自己的通讯平台。  多端统一：一套系统覆盖手机端、桌面端和 Web，满足随时随地沟通协作。 功能完整：支持文本、图片、图文、视频、文件、语音、名片等丰富消息形态。 社交链路成熟：好友申请、加好友验证、黑名单、群聊创建、邀请入群、入群审核、群设置等能力完整,不仅有消息收发，还有引用回复、转发、撤回、删除、置顶能力。
